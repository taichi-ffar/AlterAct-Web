# AlterAct Web

Unity製カードゲーム『AlterAct』プロトタイプのブラウザ版です。

## Play

[AlterActをブラウザでプレイ](https://taichi-ffar.github.io/AlterAct-Web/)

初回起動時はゲームデータの読み込みに少し時間がかかります。PC版のChromeまたはEdgeに加え、スマートフォンの縦画面と全画面表示に対応しています。

手札のソート、同じ手番中の操作巻き戻し、ヘイゼの返却カードを手札から選ぶ操作を利用できます。

## Build

- Unity `6000.5.4f1`
- WebGL / WebAssembly
- Gzip compression with decompression fallback

このリポジトリには公開用WebGLビルドのみを収録しています。
